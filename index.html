Games e Animações

Nós aprenderemos a desenvolver jogos 2D, aprendendo desde a concepção do tema do jogo, passando pelo seu funcionamento, analisando as regras do jogo e desenvolvendo toda a parte visual de animação, criação dos personagens e a programação para o funcionamento do jogo. Poderão desenvolver efeitos especiais nas animações, aprenderão a física de funcionamento dos personagens, integração com multimídias, tudo com uma aplicação que funcione através da internet, como o Scratch e depois passaremos para o AppInventor.

Programação em Python

Neste curso, vamos estudar a linguagem de programação mais querida dos desenvolvedores na atualidade. Com Python, atualmente podemos trabalhar desde aplicações que manipulam informações simples, como dados de hardware, até informações complexas de alto poder de processamento, como ações dentro de games. Python é uma linguagem em ampla e constante evolução e adesão para aplicações de pequeno, médio e grande porte.
